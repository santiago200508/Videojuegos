Sobre GameStore

Proyecto académico de desarrollo web

Acerca del Proyecto

GameStore es un proyecto académico desarrollado como parte de mi formación en desarrollo web. Esta plataforma simula una tienda en línea de videojuegos, implementando las mejores prácticas de diseño y desarrollo web moderno.

El proyecto fue construido utilizando Angular como framework principal y Bootstrap para el diseño responsive, garantizando una experiencia de usuario óptima en todos los dispositivos.

A través de este proyecto, he aplicado conocimientos de HTML, CSS, TypeScript, componentes reutilizables, enrutamiento y diseño responsive.
